
>> Pacotes necessários

sudo trizen -S pulseaudio-alsa
sudo trizen -S pulseaudio-bluetooth
sudo trizen -S bluez
sudo trizen -S bluez-libs
sudo trizen -S bluez-utils

Verificar se o bluetooth está bloqueado
sudo rfkill list

No caso de estar bloqueado, basta executar:
rfkill unblock bluetooth ou wifi

bluetoothctl

[bluetooth]# agent on
[bluetooth]# default-agent

$ pulseaudio -k

[bluetooth]# scan off
